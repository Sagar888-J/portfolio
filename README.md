# Sagar - Full Stack Developer Portfolio

A premium, modern, and fully responsive portfolio website built for a Full Stack Developer.

## 🚀 Features

- **Premium Modern UI**: Glassmorphism effects, gradient backgrounds, and deep dark theme.
- **Interactive Elements**: Typing effect, scroll animations, animated statistics counters.
- **Project Filtering**: Dynamic sorting of portfolio projects.
- **Responsive Design**: Mobile-first approach, works seamlessly across all devices.
- **Clean Code Architecture**: Semantic HTML5, Vanilla CSS3 with variables, and Modular ES6+ JavaScript.

## 🛠️ Technology Stack

- **HTML5**: Semantic structuring
- **CSS3**: Custom variables, Flexbox, Grid, Animations
- **JavaScript (ES6+)**: DOM manipulation, Intersection Observer API
- **FontAwesome**: Icons
- **Google Fonts**: Inter & Outfit

## 📁 File Structure

```text
portfolio/

├── css/

├── js/

├── assets/             # Place your assets here
│   ├── images/
│   ├── icons/
│   └── resume/
└── README.md
```

## ⚙️ How to Use

1. Clone or download the repository.
2. Replace the placeholder images in the `assets/images` directory.
3. Add your actual resume to `assets/resume/Sagar_Resume.pdf`.
4. Open `index.html` in any modern web browser to view the portfolio.
5. Deploy to any static hosting provider (e.g., GitHub Pages, Vercel, Netlify, Hostinger).
